# User Personas — Phase 10 Web Application

---

## Persona 1: Casual Carlos

**Type**: Casual Player

**Demographics**:
- Age: 25-45
- Tech comfort: Moderate (uses smartphone daily, familiar with web apps)
- Gaming experience: Plays card/board games occasionally with friends and family

**Goals**:
- Play a quick game of Phase 10 with a friend online without installing anything
- Learn the rules easily if unfamiliar with Phase 10
- Have fun without needing to memorize complex strategies

**Behaviors**:
- Opens the game on mobile during breaks or in the evening
- Prefers simple, intuitive interactions (tap to select, obvious buttons)
- May not finish a game in one session — appreciates being able to resume
- Plays against AI when no friend is available

**Frustrations**:
- Confusing UIs that don't explain what to do next
- Having to create accounts or install apps for a simple card game
- Accidentally making irreversible moves
- Small touch targets on mobile

**Relevant Stories**: Core Gameplay, Game Management (persistence, undo), UI/UX, AI Opponents

---

## Persona 2: Competitive Mia

**Type**: Competitive/Experienced Player

**Demographics**:
- Age: 20-35
- Tech comfort: High (developer, gamer, or tech-savvy professional)
- Gaming experience: Knows Phase 10 rules well, plays regularly

**Goals**:
- Play strategically and track performance over multiple games
- Challenge herself against Hard AI when friends aren't available
- Create custom phase lists to try different game variations
- Quick game setup — no friction to start playing

**Behaviors**:
- Plays primarily on desktop/tablet for better visibility
- Wants to see game history and scores
- Uses keyboard shortcuts when available
- Hosts multiplayer games and shares links with friends

**Frustrations**:
- Slow AI that takes too long to "think"
- Inability to customize game rules or phase orders
- No way to review past game performance
- Poor responsiveness or laggy animations

**Relevant Stories**: Core Gameplay, Multiplayer (hosting), AI Opponents (hard mode), Custom Phase Lists, Game Management (history)
